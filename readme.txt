== Description ==

Extends ACF to the post type objects themselves. It is a great way to add meta data such as description or header image for a post type.



== Installation ==

This software can be treated as both a WP plugin and a theme include.

1. Copy the 'acf-post-type-options' folder into your plugins folder
2. Activate the plugin via the Plugins admin page
